# E-Commerce Project

A full-stack e-commerce application built with React frontend and Express.js backend, featuring user authentication, product management, shopping cart, and order processing.


## ğŸŒŸ Key Features
### Frontend
- User authentication (signup, login)
- Product browsing and search
- Shopping cart management
- Checkout process
- User profile management
- Admin panel for adding products without authication
- Responsive design with Tailwind CSS
- Real-time search with debouncing

### Backend
- RESTful API with Express.js
- JWT-based authentication
- CRUD operations for products
- Cart and order management
- File uploads with multer integration
- MySQL database
- Password hashing with bcryptjs




## Tech Stack

### Frontend
- React 19
- Vite (build tool)
- Redux Toolkit (state management)
- React Router (routing)
- Tailwind CSS (styling)
- Axios (HTTP client)
- Heroicons & Lucide React (icons)

### Backend
- Node.js
- Express.js
- MySQL2 (database)
- JWT (authentication)
- bcryptjs (password hashing)
- Multer (file uploads)
- CORS (cross-origin requests)

## ğŸ“¦ MySQL Tables

-- Users
CREATE TABLE e_commerce.users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  email VARCHAR(100) NOT NULL UNIQUE,
  password VARCHAR(255) NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Products
CREATE TABLE e_commerce.products (
  id int NOT NULL AUTO_INCREMENT,
  name varchar(255) DEFAULT NULL,
  price decimal(10,2) DEFAULT NULL,
  description` text,
  stock int DEFAULT NULL,
  image varchar(255) DEFAULT NULL,
  created_at timestamp NULL DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (id)
);

-- Cart
CREATE TABLE e_commerce.carts (
  id INT AUTO_INCREMENT PRIMARY KEY,
  user_id INT UNIQUE,
  CONSTRAINT fk_carts_user
    FOREIGN KEY (user_id)
    REFERENCES users(id)
    ON DELETE CASCADE
);

-- Cart Items Table
CREATE TABLE e_commerce.cart_items (
  id INT AUTO_INCREMENT PRIMARY KEY,
  cart_id INT NOT NULL,
  product_id INT NOT NULL,
  quantity INT DEFAULT 1,
  FOREIGN KEY (cart_id) REFERENCES carts(id) ON DELETE CASCADE,
  FOREIGN KEY (product_id) REFERENCES products(id) ON DELETE CASCADE
);

-- Orders
CREATE TABLE e_commerce.orders (
  id INT AUTO_INCREMENT PRIMARY KEY,
  user_id INT NOT NULL,
  total DECIMAL(10,2) NOT NULL,
  name VARCHAR(100),
  address varchar(255),
  phone VARCHAR(20),

  CONSTRAINT fk_orders_user
    FOREIGN KEY (user_id)
    REFERENCES users(id)
    ON DELETE CASCADE
); 

-- Order Items
CREATE TABLE e_commerce.order_items (
  id INT AUTO_INCREMENT PRIMARY KEY,
  order_id INT NOT NULL,
  product_id INT NOT NULL,
  price DECIMAL(10,2) NOT NULL,
  quantity INT NOT NULL,

  CONSTRAINT fk_order_items_order
    FOREIGN KEY (order_id) REFERENCES orders(id) ON DELETE CASCADE,

  CONSTRAINT fk_order_items_product
    FOREIGN KEY (product_id) REFERENCES products(id) ON DELETE CASCADE
);
